Project Description: WallShooter Homework assignment 3 for Unity Game Dev class at Florida State Unversity. Project
consists of 3 scripts: WallGenerator, Shooter, and CollisionDetection. The player can move with the UP,DOWN, LEFT, and RIGHT
keys and shoot a ball with the SPACE bar. When the ball hits the bricks in the wall, they will turn yellow due to
the collision detection script on the ball.
